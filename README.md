## How to run

**Native way**:

```markdown 
mvn clean install  
mvn spring-boot:run
```

**Docker way**:

REQUIRED: docker-compose (https://docs.docker.com/compose/install/), tested under 2.1 docker-compose version

```markdown
docker-compose up -d
```

in the project root
